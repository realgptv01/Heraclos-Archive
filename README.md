# Heraclos Archive
Welcome! This is the Heraclos Archive! This repository aims to archive the entirety of the Heraclos game files. 

# How I did this
Heraclos works by fetching game assets from https://heraclosgame.com. If you use Chrome DevTools' Network feature, you can see those assets getting fetched in real time. You can also download these files by clicking on them.
